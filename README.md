# MERN Stack - Vehicle Management Dashboard

# How to run the application

## Backend

To deploy an application run the following commands:

```
cd backend
npm install
npm run start
```

## Frontend

To run a client application run the following commands:

```
cd client
npm install
npm run dev
```
